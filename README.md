# MARQUEE_MERN

A collection of front-end practice exercises covering core HTML and CSS concepts. Despite the repo name, this currently contains HTML/CSS learning exercises rather than a full MERN (MongoDB, Express, React, Node.js) application — it looks like the starting point for a broader MERN learning journey.

## 📁 Project Structure

```
MARQUEE_MERN/
├── Semantic Elements/     # Practice with HTML5 semantic tags (header, nav, section, article, footer, etc.)
├── Table/
│   └── PracticeSet/       # Exercises building and styling HTML tables
├── forms/
│   └── PracticeSet/       # Exercises building HTML forms and input elements
├── handson/                # Hands-on coding exercises
└── list/                    # Practice with ordered/unordered/definition lists
```

## 🚀 Getting Started

No build tools or dependencies are required — these are static HTML/CSS files.

1. Clone the repository:
   ```bash
   git clone https://github.com/immeetparmar/MARQUEE_MERN.git
   cd MARQUEE_MERN
   ```
2. Open any `.html` file directly in your browser, or use a live-reload tool such as the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) VS Code extension for a smoother development experience.
